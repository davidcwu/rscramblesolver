# Rscramblesolver

A word finder for Scramble-related games

## Installation

Add this line to your application's Gemfile:

    gem 'rscramblesolver'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rscramblesolver

## Usage

Run 'rscramblesolver' from the command line.
